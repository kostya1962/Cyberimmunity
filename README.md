## Постановка задачи

Необходимо разработать акхитектуру для web-приложения (далее ПО - программное обеспечение) для книжного интернет-магазина, задача которого: эффективно управлять каталогом книг, позволяя клиентам выбирать и покупать необходимые книги. Web-ресурс должен включать как десктопную, так и мобильную реализации, а также базу данных, хранящую персональные данные клиентов.
